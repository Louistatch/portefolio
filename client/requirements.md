## Packages
react-helmet-async | For SEO optimization and meta tags
react-markdown | For rendering MDX/Markdown in blog posts
remark-gfm | For GitHub Flavored Markdown support (tables, strikethrough)
rehype-raw | To allow raw HTML in markdown if needed
date-fns | Formatting dates for blog posts and appointments
